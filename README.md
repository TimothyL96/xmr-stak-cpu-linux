# Xmr-stak-cpu-linux
### Monero mining software on Linux
#### No donation

#### Download with linux command: 
`wget https://raw.githubusercontent.com/ttimt/xmr-stak-cpu-linux/master/run.sh`

`chmod +x run.sh`

`sudo ./run.sh`

##### The download will be at xmr-stak-cpu-linux-master
##### Run with
`sudo ./xmr-stak`
